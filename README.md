# NES emulator ファミコンエミュレーター
This is a Famicom emulator created using client-side HTML JavaScript.<br>
これはクライアントサイド HTML JavaScript を使用して作成されたファミコン エミュレータです。
# How to use
It's basically the same as any other NES emulator.<br>
基本的には他の NES エミュレーターと同じです。
